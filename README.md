# UK Data Service Survey Team content creation templates

This internal repository contains the source code of a multi page  **book** Quarto project in html format. It follows UK Data Service guidelines and has been approved by Comms.
It includes an example R page, and example Stata page and basic instructions on how to create your own documents using this one as template. 

A brief tutorial on how to use this template to create your document as well as the rendered book are available on [GitHub Pages](https://special-succotash-jzz4pqz.pages.github.io/)
